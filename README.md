# 卖座网

> 使用vue + vuex开发的卖座网

### 已经实现功能

1.首页（侧边栏、点击进入电影详情界面）

2.通讯录（联系人字母排序、搜索界面）

3.发现（朋友圈）

4.我（界面）

### 待实现功能（接下来一个月陆续完成）

1.语音搜索、发送短视频、地理位置等

2.朋友圈细节完善

3.扫一扫

4.相册、钱包

5.搭建服务器实现实时通信功能以及其他细节实现

## 该项目技术栈

> webpack + vuejs + vuex + vue-resource + vue-router + sass
